# ☁️ CloudTask Manager

A simple and responsive CRUD Task Manager built using **Node.js**, **Express.js**, and **IBM Cloudant NoSQL Database**.

## 🚀 Features

- ➕ Add Tasks
- 📋 View Tasks
- ✏️ Update Tasks
- 🗑️ Delete Tasks
- 🔍 Search Tasks
- 📊 Task Counter
- ☁️ IBM Cloudant Integration
- 📱 Responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- IBM Cloudant NoSQL Database

## ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

Open

```
http://localhost:5000
```
